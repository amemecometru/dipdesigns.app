Quickstart

Go from zero to your first live UI in under a minute.

0. Configure Your API Key

0

Before you can generate, you need an API key. Get a free OpenRouter key at openrouter.ai/keys, then paste it here:

1. Generate Your First UI

1

Navigate to the App tab. Type a prompt like a dark dashboard with a sidebar and a chart and press Enter.

2

Watch Gemma generate the HTML/CSS/JS live in the preview pane. Toggle between Desktop and Mobile views.

2. Use Skills from the Library

3

Open the Library tab. Browse community skills like "Dark Dashboard" or "Landing Page."

4

Click Use Skill to auto-fill the prompt and generate instantly.

3. Sync to Desktop (Premium)

5

Run node receiver.js on your laptop. Set the desktop IP in the PWA console:

WebhooksEmail.setDesktopIP('192.168.1.42')
6

Click Send to Desktop — the generated file lands in your project folder instantly.

4. Webhook Ingress (Premium)

Send a prompt from any external service (email, GitHub, Cloudflare, etc.) to your webhook endpoint:

POST /api/webhook
{ "prompt": "a landing page with a hero section", "session": "my-session" }

The Worker processes it, broadcasts via SSE, and your PWA renders it live.
